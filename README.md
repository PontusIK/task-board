# Task-board Demo

Spring boot task-board

## Running application

### Requirements

* Java-21+
* Docker

### Build

1. fetch source code:\
    Clone this repository with git or download as zip and extract.

2. build:
    Linux/MacOS:
    ```
    gradlew build
    ```

    Windows:
    ```
    .\gradlew.bat build
    ```

### Start

with docker engine running:
```
docker compose up
```
